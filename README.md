# GacaKiller   <small>MIT</small>

一键恢复被 **伽卡他卡电子教室** 隐藏的文件。  
One-click restore files hidden by Gacaka classroom software.

---

## 快速开始 / Quick Start

1. 把 `GacaKiller.bat` 放进 U 盘 **根目录**  
2. 右键 → **以管理员身份运行**  
3. 脚本会自动：  
   - 去掉隐藏属性  
   - 把被缓存的原文件移回根目录  
   - 删除 `._cache_xxx` 空壳  
4. 拔插 U 盘，文件重新出现！

---

## 常见问题 / FAQ

| 现象 | 原因 | 解决 |
|---|---|---|
| 只有 `._cache_原文件名` 且 0 KB | 伽卡他卡缓存机制 | 运行本脚本 |
| 运行后仍无文件 | 可能已被 FOUND.000 碎片化 | 用 CHK 还原工具 |
